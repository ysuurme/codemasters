# CodeMasters

> **CodeMasters offers skills of the future to those who need it most.**
> During a 10-week program, we support participants in learning how to code
> and help them grow towards a career in the Netherlands.

---

## What is in this repository?

This repository contains all the course material for the CodeMasters bootcamp programs.
You will find interactive notebooks, runnable scripts, coding examples, and assignments here.

---

## How to navigate this repository

| Folder | What you will find |
|--------|-------------------|
| [fundamentals/](fundamentals/) | Fundamentals Program — 10 weeks for beginners |
| [advanced/](advanced/) | Advanced Program — 10 weeks of deeper topics |
| [graduation-projects/](graduation-projects/) | Example projects and template for your graduation project |

---

## How to run the examples

From this folder, run:

```
python main.py
```

A menu will appear where you can pick an example to run.

---

## What does each week look like?

Every week folder has the same structure:

```
week-NN-topic-name/
├── README.md        Start here — what you will learn this week
├── notebooks/       Interactive notebooks (run with: marimo run filename.py)
├── scripts/         Python scripts you can run by pressing F5
└── assignments/     Your weekly assignments (read assignments/README.md first)
```

---

## Getting started

1. Make sure Python is installed: `python --version`
2. Install Marimo: `pip install marimo`
3. Go to your current week folder and read the `README.md` inside
4. Run a script with F5 in your editor, or type `python scripts/filename.py` in the terminal
5. Open a notebook with `marimo run notebooks/filename.py`
