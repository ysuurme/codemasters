# Week 02 — Assignments

Three assignments to practise what you learned this week.
Open each `.py` starter file, read the instructions inside, and complete the tasks.

---

## Assignment A — Personal introduction

**File:** `assignment_a.py`

Fill in your own details (name, birthdate, height, favourite food, siblings)
and use `print()` with f-strings to introduce yourself in a short paragraph.

Python calculates your age automatically from the birthdate — no manual updating needed.

**You must use variables** — do not hard-code the values directly into the print statements.

---

## Assignment B — Type conversion exploration

**File:** `assignment_b.py`

Explore what happens when you convert between types:
- Converting strings to `int` and `float`
- What happens when you try `int("hello world")` — read the error and explain it
- The truncation trap: `int(3.9)` is not `4`

Answer the reflection questions as comments in the file.

---

## Assignment C — Christmas tree 🎄

**File:** `assignment_c.py`

Print a Christmas tree in the terminal using only `print()` and string operations.

Use the `*` operator to repeat characters — do not count spaces by hand.

**Requirements:**
- At least 6 layers of branches
- A trunk at the bottom
- Use string repetition, not manual spacing

**Extra points:**
- Add a star or decoration at the top
- Add ornaments on the branches
- Print a second object (snowman, house, rocket — anything!)

---

## How to run

From the project root:

```sh
python fundamentals/week-02-python-syntax/assignments/assignment_a.py
python fundamentals/week-02-python-syntax/assignments/assignment_b.py
python fundamentals/week-02-python-syntax/assignments/assignment_c.py
```

When you are happy with your work, commit it with a short message, for example:
`"Week 02: completed all three assignments"`
