# Fundamentals Program

Welcome to the CodeMasters Fundamentals Program!

This is a 10-week course designed for people who are completely new to programming.
No prior experience needed — we start from zero and build up step by step.

## How to use this program

1. Go to the folder for your current week
2. Read the `README.md` inside that folder
3. Open and run the notebooks or scripts
4. Complete the assignment at the end of the week

Each week builds on the previous one, so follow the order.
