# Graduation Projects

At the end of the CodeMasters program, every participant creates their own programming project.
This is your chance to apply everything you have learned and build something you are proud of.

---

## How to get started

1. Look at the `examples/` folder for inspiration
2. Copy the `template/` folder and give it your own project name
3. Read `template/README.md` for guidance on how to structure your project
4. Build your project and prepare to present it!

---

## Folders

| Folder | What you will find |
|--------|--------------------|
| [examples/](examples/) | Finished example graduation projects for inspiration |
| [template/](template/) | A starter template to copy for your own project |
